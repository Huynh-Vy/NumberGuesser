# Number Guesser project

## The project is for building a small number guessing game with HTML DOM and javascript.

### How the game works:
1. There are 2 players: user and computer
2. User input number from 0 to 9
3. Click on Make a Guess button to see who is winner, the winner will be the one who guesses the number closer to target number
4. Click next round button to play again

Technologies:
* HTML
* CSS
* Javascript
